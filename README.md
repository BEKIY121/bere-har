# Bere Har — Artistic Resort Wear

A high-fashion, editorial luxury resort wear website built with **Vite + React**.

## Design Philosophy

Inspired by the editorial aesthetics of Vogue and high-end fashion magazines, the site features:
- A cream editorial canvas with massive background typography
- PNG cutout model overlays in the Vogue collage style
- Playfair Display serif + Inter sans-serif typography
- La Belle Aurore script accents
- Framer Motion animations throughout
- Multi-page routing with React Router

## Pages

- `/` — Home (Hero, Marquee, Story, Featured Look, Collection, Brand Stats, Craft, Journal, Newsletter)
- `/shop` — Full collection with editorial product grid
- `/philosophy` — Brand story, history timeline, values, founders quote
- `/atelier` — Craft process ("The Sun, The Waves, The Wind")

## Tech Stack

- **Vite** — Build tool
- **React 19** — UI framework
- **Framer Motion** — Animations
- **React Router DOM** — Page routing
- **Lucide React** — Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
